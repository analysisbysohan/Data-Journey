# Data Analyst Journey 📊

Welcome to my data analytics learning repository! This project tracks my daily progress as I master Python, SQL, and data visualization tools to solve real-world problems.

## 🚀 What's Inside?

### 🐍 Python Modules & Packages
* **`basic_calculator.py`**: A clean mathematical module handling core operations, including recursion for factorials and custom error handling for division by zero.
* **`area_calculator.py`**: A geometric module built to compute the area of shapes like circles, squares, rectangles, and triangles.
