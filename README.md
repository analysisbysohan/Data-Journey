# Data Analyst Journey 📊

Welcome to my data analytics learning repository! This project tracks my daily progress as I master Python, SQL, and data visualization tools to solve real-world problems.

## 🚀 What's Inside?

### 🐍 Python Core Programming
* **`basic_calculator.py` / `simple_calculator.py`**: Functional, terminal-based calculator scripts demonstrating modular functions (`def`), user input type-casting, recursion for factorials, and conditional control flows.
* **`area_calculator.py`**: A geometric module built to compute the area of shapes like circles, squares, rectangles, triangles and SURFACE AREA OF cylinder.
* **`conditionals.py`**: A foundational script demonstrating user input handling, type casting (`int`), and conditional control flow (`if-else`) to compare two integers and output the maximum value.
* **`simple_interest.py`**: A financial calculation script that takes user inputs for Principal, Time, and Rate to compute and display Simple Interest using dynamic functions.
* **`compound_interest.py`**: A financial script that calculates compound interest using the geometric `pow()` function, featuring user input handling and floating-point precision formatting.
